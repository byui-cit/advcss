Layout: Get out of your Ruts!

The last few years have been really exciting for web designers/developers. With the arrival and support for things like Flexbox, CSS Grid, and others we finally have not just good...but great tools to accomplish our layouts.

Suddenly things that we not possible (or at least things that involved a lot of pain to do) are not only possible but in some cases easy to do. Lets take this opportunity to get out of ruts and experiment with these new tools.

Activity

Step 01:  Begin this activity by watching this great presentation by Jen Simmons on Designing for Grid (~1h10 long).

This presentation (optional) is also very good. It spends more time with code.

Step 02:  Once you are done spend a little time looking at the example of printed design below. Choose one side or the other and think about how it might translate into a webpage. Think in particular about how you would reformat it to work on a mobile sized screen. Create a wireframe showing how you would present the content.

Step 03: Now create the mobile version of your layout. After you have the mobile one working, move to the desktop version. You can use whatever images you would like for the site.

Here are a few resources that might help while creating your poster.

  • MDN Clip-path
  • MDN Writing Mode
  • Clippy CSS clip-path maker
  • Lorem Ipsum generator
  • Lorempixel: placeholder image service

Step 05: Upload your layout, image, and wireframe to iLearn.

Grading

  • Mobile Wireframe produced for a print layout. 5pts
  • Responsive layout created that matches print example. 10pts
