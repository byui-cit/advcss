Layout: Get out of your Ruts!

The last few years have been really exciting for web designers/developers. With the arrival and support for things like Flexbox, CSS Grid, and others we finally have not just good...but great tools to accomplish our layouts.

Suddenly things that we not possible (or at least things that involved a lot of pain to do) are not only possible but in some cases easy to do. Lets take this opportunity to get out of ruts and experiment with these new tools.

Activity

Step 01:  Begin this activity by looking for inspiration. Watch this great presentation by Jen Simmons on Designing for Grid (~1h10 long).

This presentation (optional) is also very good. It spends more time with code.

Step 02:  Once you are done spend a little time looking at examples of printed design. This could be in magazines, posters, books, etc. Choose one page layout that you would like to convert to HTML/CSS.

Here is a few ideas to get you started on your search if you need them. Find something that you think will push you a bit...but don't go too complex :)

  • Jules Forrest Grid Experiments
  • TutsPlus 20 Magazine Layouts
  • Pinterest
  • Editorial Template

Step 03:  Take your selected layout and think about how you would present it on a mobile device. Create a wireframe showing this.

Step 04: Now create the mobile version of your layout. After you have the mobile one working, move to the desktop version.

Step 05: Upload your layout, image, and wireframe to iLearn.

Grading

  • Mobile Wireframe produced for a print layout. 5pts
  • Responsive layout created that matches print example. 10pts
