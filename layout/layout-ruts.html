Layout: Get out of your Ruts!

The last few years have been really exciting for web designers/developers. With the arrival and support for things like Flexbox, CSS Grid, and others we finally have not just good...but great tools to accomplish our layouts.

Suddenly things that we not possible (or at least things that involved a lot of pain to do) are not only possible but in some cases easy to do. Lets take this opportunity to get out of ruts and experiment with these new tools.

Activity

Step 1

Begin this activity by watching this great presentation by Jen Simmons on Designing for Grid (~1h10 long).

This presentation (optional) is also very good. It spends more time with code.

Make sure that you have also read Chapter 1 of Atomic Design, and reviewed a bit about making fully responsive images using the picture element

Step 2

Once you are done spend a little time looking at the example of printed design below and think about how it might translate into a webpage.

Consider this design through the lens of the components introduced in the Atomic design chapter. Can you see the pattern with the block of content? Each is constructed of an image, a title, and description.

Step 3

Lets create a media component that we can use to display those blocks of content. Your component should have the following characteristics:

  • All of the CSS code to create the component should reside in a SCSS partial.
  • The images in the component should be fully responsive. Not only resizing visibly, but the browser should have access to different sized image files and pull the correct file for the viewport size. (picture element, srcset, source)
  • The images should also be able to change shape if necessary at different screen sizes (art direction)
  • Keep the HTML as simple as possible to make it portable.
  • The information block should be easily movable: top or bottom, and left or right.
