Create a Walkthrough

It's said that one of the best ways to learn something is to have to teach it. This week you had the opportunity to learn about creating shapes with CSS. To help the concepts you were introduced to this week stick, I would like to give you the opportunity to teach someone else about them. You need to create a walkthrough instructing how to create a specific shape with css and borders.

Activity

Step 1:   Complete the Shapes with CSS activity then select a shape to base your walkthrough on. You might refer back to the CSS-Tricks: Shapes of CSS article.

Step 2:   Prepare your code and script for your walkthrough then create a screencast of you teaching the material you have prepared. Post your screencast to the Class Discussion board. If you don't have some screencast creation software that you already use I've had pretty good success with Screencast-o-matic

Grading

  • Screencast shows understanding of the topic. (7pts)
  • Screencast includes clear examples. (7pts)
  • Student narration is clear and understandable. (6pts)
