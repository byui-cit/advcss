Converting CSS to SCSS

Activity

During the first week of the semester you were asked to produce the css for a website. This week we will take that css and convert it to scss.

Step 1:

Review the css you produced for the review website to see where SCSS might be helpful. Add variables, nesting, and mixins where appropriate. You must however use each of those at least once.

Step 2:

Upload your .scss file to the dropbox for this activity in Ilearn.

Grading

  • Sass variables used correctly where appropriate. (5pts)
  • At least one example of nesting is used. (5pts)
  • At least one mixin is created and used correctly. (5pts)
