Lesson Focus

This week's focus is to:

  1. learn what CSS preprocessors are and what they can do for us,
  2. learn the differences between 2 of the major preprocessors: Less and Sass
  3. learn the basic syntax for SCSS,
  4. learn how to convert SASS to CSS,
  5. and gain experience with SASS by converting an existing stylesheet from CSS to SCSS.

Assigned Readings

  • CSS Master: Ch8. Using CSS with SVG on Sitepoint
  • Youtube: Sass and Compass
  • Step by Step Sass course on Sitepoint (Optional additional resource for Sass...good 1h15 long course, but warning, there is some imagery in here that some might find distracting.)
  • Jump Start Sass book on Sitepoint (Another optional additional Sass resource. Chapters 3-4 are particularly pertinent this week.)
