Other Libraries

Foundation is just one of many CSS frameworks/libraries. Lists like this top 21 css frameworks are pretty common to try to help developers keeps tabs on all of them. Lets review a just a few that you should be aware of.

Watch for these things:

  • Modernizr: Runs a series of tests as the page loads to determine what features of html and css are supported in the current browser. Using the results of that can allow developers to deliver tiered experiences.
  • Normalize: Browsers come with different default styles for common html elements. Normalize helps bring these together for a more consistent experience.
  • Compass and Bourbon: Both of these have collections of commonly used Sass mixins and partials. Compass is probably a bit more full featured, Bourbon is lighter weight
  • Bootstrap: Another full framework similar to Foundation.
  • Material Design: the link I provided is not to a framework or library. Material Design is actually a set of specifications about how a Google app should be styled. It has become quite popular, to the point that there are many css frameworks that have been built to aid in implementing it.
  • UIkit: This is a collection of css and js to implement ui elements and interfaces quickly.
