CSS Module: Mastering Custom Properties

Hours: 2

Custom Properties are a long awaited addition to CSS. They allow for the setting of variables right in the CSS. One of the most exciting features is that these variables can be changed at any time with by either CSS override or Javascript.

Prepare

Review the following resources on CSS Custom Properties

  1. Using CSS custom properties. MDN
  2. A Strategy Guide To CSS Custom Properties Smashing Magazine

Ponder/Prove

Complete the following activity:

Imagine you have been asked by a client to produce a prototype of a web component that includes a headline, paragraph, and image. The headline and paragraph typeface should look noticably different but size and download speed is a major concern for this project. You can only use one font.

Google Fonts has added support for Variable Fonts and has several free ones to choose from. Select one and then use it to produce your prototype. For the most flexibility you should download the font and define your own @font-face instead of using the Google Fonts generated CSS.

Take advantage of the resources in the articles to help you discover how many different looks your font might have.
