Hours: 6

Objectives

  1. Improve our ability to learn how to learn.
  2. Improve our mastery of CSS:
    • Learn the syntax for working with CSS border radius and CSS gradients.
    • Learn what can be accomplished with border radius and gradients.
    • Successfully apply your new knowledge.

Prepare

As you review the following resources on creating CSS shapes and Gradients keep in mind our objectives. Notice that one of those is not "spend x minutes reading" or "read x number of articles". Instead our goal is to gain some specific knowledge and skills.

Part of becoming better at learning is recognizing how much time you need to spend gathering new knowledge before you should start trying to apply your new knowledge. Too much time gathering and not enough application is bad, but so is jumping straight into application without enough time first spent building a foundation of knowledge to pull from!

Note that some of the resources here are more references (things that will be helpful to pull up as you are applying your knowledge), and others are more instructional (teach you how to use the technologies). You should spend most of your prepare time on the instructional resources. Essentially, study these resources until you feel comfortable with the concepts, then go try it!

  1. Review: CSS Tricks: The Shapes of CSS
  2. Review: MDN: Border Radius
  3. Read: Web.Dev: Gradients
  4. Read: Smashing Magazine: CSS radial and Conic Gradients
  5. Review: CSS Tricks: Complete guide to Gradients
  6. The Humble Border-Radius
