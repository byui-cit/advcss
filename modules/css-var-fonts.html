CSS Module: Mastering Variable Fonts

Hours: 2

Variable Fonts are an exciting new addition to CSS. They help us solve the problem of excessive weight added to a page when several styles of custom fonts are required (bold, italic, etc).

Prepare

Review the following resources on CSS Variable Fonts

  1. Varible Fonts Guide MDN
  2. Introduction to Variable Fonts 24 Ways
  3. If you have never defined a @font-face yourself then you will also find this article on Using @font-face to be helpful as well.

Ponder/Prove

Complete the following activity:

Imagine you have been asked by a client to produce a prototype of a web component that includes a headline, paragraph, and image. The headline and paragraph font should look noticably different but size and download speed is a major concern for this project. You can only use one font.

Google Fonts has added support for Variable Fonts and has several free ones to choose from. Select one and then use it to produce your prototype. For the most flexibility you should download the font and define your own @font-face instead of using the Google Fonts generated CSS.

Take advantage of the resources in the articles to help you discover how many different looks your font might have.
