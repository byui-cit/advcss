Objectives

  1. Improve your ability to learn how to learn.
  2. Improve your CSS layout mastery:
    • Grid
    • Flexbox
    • Component based layout
  3. Improve your mastery of CSS:
    • Apply Accessibility best practices in your CSS and HTML.
    • Assess accessibility issues and fix them
  4. Improve your mastery of CSS architecture.
  5. Improve your mastery of CSS troubleshooting.
  6. Improve your mastery of SCSS.

  1. You should have completed at least ONE of the modules below before this one.
    • Building Responsive Components
    • Flexbox
    • CSS Grid
  2. Watch Introduction to Web Accessibility WebAIM
  3. Explore Wave Tool WebAIM
  4. If you haven't reviewed it before it would also be helpful to look at BEM
