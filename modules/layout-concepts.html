Layout Module: Core Concepts

Hours: 3

Prepare

This activity is a little different than most of the others. It provides preparation for all the other Layout modules, and as such will have no Prove portion of it's own. It will instead send you off to complete another layout module to get more depth in one aspect of layout. Count this module finished when you have completed the reading, then go do another Layout module as a followup.

You will notice some overlap and repetition in these resources. Much of it might feel like review. This is intentional. There are some important key concepts presented here that are critical to understand. Hopefully reading about them from multiple different sources will help them to settle in your brain.

  1. Carefully read CSS Master Ch5: Layouts (Sitepoint).
  2. Read Smashing Magazine: Getting started with CSS Layout.
  3. Watch AEA: Designing Intrinsic Layouts Jen Simmons (1hr). This presents a lot of great concepts while going pretty deep into Grid.
  4. Read Grid Theory (Sitepoint).

Optional Resources

Here are a few more articles on topics if you need a review on them.

  1. Read Smashing Magazine: Using Media Queries.
  2. Read 7 habits of highly effective media queries.
  3. Review MDN: CSS Layout.

Prove

Count this module complete when you have finished the Prepare section above, but follow it up by completing one or more of the following layout modules:

  1. Flexbox practice.
  2. CSS Grid Practice.
  3. Multi Column Practice.
  4. Building Components.
  5. (Re)build a real site.
