Layout Module: CSS Grid

Hours: 6

Prepare/Ponder

Review the following resources about CSS Grid. Note! There is a lot of material here. You should spend enough time with these resources to feel you are understanding CSS Grid. Understanding Grid is the goal...not getting through the resources.

  1. Grid By Example Rachel Andrew's great tutorial site for Grid. I would recommend as a start to watch:
    • Defining a grid (Transcript) 2:57min
    • Repeat notation (Transcript) 1:48min
    • Line based positioning (Transcript) 4:16min
      • and
    • Grid template areas (Transcript) 4:14min
    Those are all videos and will take about 20 minutes to view. Then move on to alignment next.
  2. Wes Bos: CSS Grid Course Another great free resource if you want to dive deep (4h). The 3rd and 4th videos (fundamentals 9min (Transcript) and dev tools 5.5min (Transcript)) I think you will find especially helpful.
  3. Introduction to the CSS Grid Layout Module Sitepoint article
  4. Complete Guide to Grid CSS Tricks...open this and keep it handy while learning Grid.
  5. Grid Garden Exercises to help become familiar with the Grid properties

Ponder

Pick another 4-5 video sections (between sections 5-19) from the Wes Bos course presenting features you are not familiar or comfortable with and watch them. (Most of them are around 5min long)

Prove

Complete the following:

  1. Complete 2-3 of the following sections from the Wes Bos Grid course listed above:
    • 20. CSS Grid Image Gallery
    • 22. Recreating Codepen
    • 23. Bootstrappy Grid with CSS Variables
    • 24. Responsive Website
    • 25. Full Bleed Blog Layout

You will gain more from these exercises if you watch the intro as he describes the task, then do as much as you can on your own...and then come back and finish watching the video explanation. At a minimum try and get all the grid layout done without help, then if you want to watch and follow along with the rest of the styling that is OK.
