Layout: (Re)build a real site: part B

Make sure that you have completed the first part of this activity!

Activity

  1. Component Styling

    In the first part you were asked to create the HTML to represent three components. Continue your work on this site by now styling those three components. Have a goal of keeping your SCSS markup as simple as possible, while focusing on re-use.

  2. Mobile Layout

    Once you have your components built and styled, write the rest of the CSS for the page to begin your styling for the mobile layout.

    As you are building and styling your page, it is ok to look at the existing page source to get ideas about how it was built. Resist the tempation to copy and paste anything from the actual page however. You will learn much more from this activity if you write all of the code yourself! You will probably also save time in the long run by writing your own code instead of trying how to get someone elses CSS code to work with your HTML.

  3. Desktop Layout

    Once you have the mobile layout done continue on to the desktop.

    If the desktop layout is giving you troubles here is a screenshot of the grid for my version of this assignment. AFTER you have tried to do it on your own feel free to look at how I setup my grid to accomodate the University Message box that is in different positions on mobile and desktop.

  4. Submit

    Upload your HTML and SCSS (zip it up) to iLearn.

Rubric

  • BYU-Idaho Home page re-created. (Everything but with the simplified navigation bar and no footer.)
  • Site is styled with SCSS and is responsive.
  • Attention paid to detail (Pixel perfect replication is not required here. It should be close but we don't have the time to make it exact.
