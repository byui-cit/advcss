Building a Simple Component

Now that you have seen one example of creating a simple HTML component, practice by creating another on your own.

Activity

  1. Review the layout.

    Below is a screenshot of the component we would like to build. It is a simple shopping cart. Review it considering how the HTML should be structured, and how you might layout the different elements.

  2. Write the HTML

    Create a new Codepen and write the HTML. Keep in mind semantics as you choose the elements to represent each part of the cart. Also keep in mind how you should structure things to make your layouts work. i.e. if you are using Flexbox or Grid do all of the right elements share a common parent?, etc. Add classes to your block and elements according to the BEM naming convention.

  3. Style

    Add your CSS/SCSS now to replicate the screenshot as closely as possible. Pay attention to padding, margins, font-sizes, placement, etc. You can just use helvetica, arial, san-serif for the font.

Completion

  • Screenshot replicated closely in HTML and SCSS.
  • Attention paid to semantics in HTML.
