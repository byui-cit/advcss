CSS Animation teaching notes

Why animation?

I would begin by reviewing the Material Design animation clip.

History of Animation

Talk briefly about animation principles: keyframes, in between frames (tweens). An animation is really just taking a change from a beginning keyfrmae to and ending keyframe and have it happen over time instead of instantly. this grid theory article.

CSS Transition

I usually just use these boxes to demo. Start by transitioning the red box doubling in size. Use SCSS where appropriate. transition property goes in the base state.

Have the students work out making one of the boxes move left and disappear. Talk about what they tried, bring up that not all properties are animatable.

Animations

If we want to do something more complex like a bouncing ball, we need more than transition. Animations allow for multiple keyframes.

Demo bouncing ball. Make sure to bring up that the entire state for each keyframe need to be defined...you need to specify what should stay the same...as well as what should change.
