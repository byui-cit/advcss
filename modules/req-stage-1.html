Prepare

The Css Zen Garden is a project started 20 years ago to show the power and potential of CSS in web design. It's premise was simple: if you take a single html file, how many different ways could it be styled without modifying the html.

The project is mostly dorment now, but a new site promises to rekindle that spirit called Style Stage. Over the course of the semester we will be creating our own versions of the Style Stage as a group project.
