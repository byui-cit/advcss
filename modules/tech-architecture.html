Earlier in the course you were asked to learn about Atomic design and implement it in the Style Guide you created for your ZenGarden project. Atomic design is one architectural design philosophy. The CSS Architecture course we began above will introduce another one: ITCSS

Both are valid, but take different approaches to the problem of organizing CSS. Complete Lesson 5 in the Sitepoint CSS Architecture course to see ITCSS in action. As you do, think back to what you learned about Atomic CSS...how do they compare? Which do you prefer? Develop an opinion!
