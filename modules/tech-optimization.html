WDD 331

Tech Module: Optimization

Hours: 3

Prepare

Review the following resources on optimizing web pages.

  1. Study Jump Start Web Performance Chapters 1-2 (Sitepoint).
  2. Review the resources at MDN: Web Performance pay special attention to the sections on Perceived Performance and CSS Performance.
  3. Watch Selector Efficiency and Simplification from the Architecture course on Sitepoint (14 min). If you are interested in the rest of that course check out the Architecture Module.

Ponder/Prove

Choose one of the testing tools from the Jump Start Web Performance book. (The Lighthouse tool in Chrome is actually a great place to start if you are unfamiliar with it.) Run the tool against the BYU-Idaho website and review the results. Make sure you understand each of the issues it brings up and do enough research to know how you might go about correcting the issue. Write up your recommendations for improving the performance of the site.
