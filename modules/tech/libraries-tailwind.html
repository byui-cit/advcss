Building your first website with TailwindCSS

Objective

Create a website that is mobile and desktop friendly using the Tailwind framework.

Activity

Please watch and follow along with the Build a Website with Tailwind CSS | Project Tutorial tutorial on Youtube. The video uses some tools that require NodeJS. You should have installed that at the beginning of the course.

Watch for these things:

  • If you still have the Tailwind installation we did earlier, you can actually just use that for this tutorial...it is basically the same. The video does structure the files slightly differently, but the rest is the same.
  • If you have not yet done so...install the "Tailwind CSS Intellisense" extention in VSCode.
  • The video makes good use of Emmet to autocomplete your html with classes. One disadvantage of that however is that you lose the benefit of the Intellisense. I would recommend not doing what the video recommends ie: h2.text-4xl.font-bold.text-center.sm:text-5xl.mb-6.text-slate-900.dark:text-white and then hitting 'tab', but instead: h2. 'tab'. That will expand into <h2 class=""></h2> and then you can type in all of the classes with the benefit of the Intellisense.
  • Speeding up the video in this case will probably not save you any time. You will be adding many many classes and will find yourself pausing a lot if you speed up the video to catch up. You might even consider slowing it down a smidge for this one.
  • The Tailwind documentation can be found at tailwindcss.com.
  • Download the files for the tutorial. You will need to do this to get the images used. Copy the img folder to the build folder if you are following exactly along with the video or in the public folder if you are using the install we did earlier in the Unit.
