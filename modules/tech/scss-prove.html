Converting CSS to SCSS

Activity

During the first week of the semester you were asked to produce the css for a website. This week we will take that css and convert it to scss.

Objective:

Demonstrate SCSS skills using variables, mixins, and nesting by modifying your css from Week 1 into SCSS.

  1. Rename your CSS file

    Rename your css file from the review website from .css to .scss. Review the css you produced to see where SCSS might be helpful.

  2. Convert CSS -> SCSS

    Add variables, nesting, and mixins where appropriate. You must however use each of those at least once.

  3. Tips

    If you are worried about having enough css to convert to scss let me give you a couple of ideas.

    • A great place to use nesting is with pseudo-classes like :hover or :active
    • @media queries are another great place to take advantage of nesting.
    • Any colors or widths you have in your css would be good candidates for variables.

    4. Grading

    • Sass variables used correctly where appropriate.
    • At least one example of nesting is used.
    • At least one mixin is created and used correctly.
